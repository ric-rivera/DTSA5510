# California Housing Analysis: Unsupervised Learning

Applies unsupervised learning techniques to identify patterns,
groupings, and anomalies in California housing data — including
geographic clustering, dimensionality reduction, and outlier detection.

**Relevance to finance operations:** Anomaly detection methods
developed here (isolation forests, DBSCAN outlier scoring) directly
inform billing variance detection and revenue leakage identification
in operational finance contexts.

**Methods:**
- K-Means and DBSCAN clustering for regional segmentation
- PCA for dimensionality reduction and pattern visualization
- Isolation Forest and Local Outlier Factor for anomaly detection
- Silhouette scoring for cluster quality evaluation

**Dataset:** California Housing dataset (Kaggle) — 20,640 records,
geographic coordinates, income, population, housing value

**Tools:** Python · scikit-learn · pandas · NumPy · matplotlib · seaborn
